Интеллектуальный анализ данных

Сервис по интеллектуальному анализу видео и фото материалов позволяет в автоматизированном режиме решать следующие задачи:
— выявлять ситуации с несанкционированным сливом топлива путем распознавания недопустимых объектов на сцене. Для работы с сервисом по распознаванию объектов перейдите по ссылке: Распознавание объектов на видео
— распознавать автомобильные номерa. Для работы с сервисом по распознаванию марки автомобиля и автомобильных номеров перейдите по ссылке: Распознавание автомобилей и автомобильных номеров на видео

Интерактивная визуализация данных

Сервис позволяет создать интерактивные графики из данных, полученных из системы GeoCode.
Для работы с сервисом перейдите по ссылке: Построение интерактивных графиков
